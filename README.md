# David Shen Portfolio Site

A static personal website for David Shen, designed to be original rather than a copy of another student's site.

## Features
- Live clock
- Interactive monthly calendar
- Award/category filters
- Public-safe toy BACE1-style model explorer
- Molecule Catcher JavaScript mini game
- Projects, research, service, leadership, and awards sections

## How to deploy on GitHub Pages
1. Create a new GitHub repository, for example `david-portfolio`.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Go to Settings → Pages.
4. Select the `main` branch and root folder.
5. Save and wait for GitHub to publish the site.
