# David Shen Portfolio Liquid v5

A cleaner, flowy liquid-glass portfolio with app links and a full `apps/` folder.

## Files to upload for the portfolio repo
Upload everything in this folder to your portfolio repository root:

- `index.html`
- `styles.css`
- `script.js`
- `README.md`
- `apps/` folder

## Included apps
The `apps/` folder contains standalone demos:

- BACE1 QSAR Lab
- Molecule Designer
- RareDx Splicing Explainer
- Clinical Trial Criteria Parser
- MedLit Evidence Map
- ChemUnit Converter
- Biochem Flashcards
- Traffic Light Simulator

## Notes
Do not publish private lab data, internal files, or patient data. Keep demos toy/public-safe.
