# David Shen Portfolio v3

Cleaner, more professional portfolio with:

- Search panel
- Dark/light theme
- Density control
- Small clock/calendar utility rail
- Education, research, projects, skills, honors, experience, and contact
- Interactive molecule designer with drug-likeness and BACE1-fit heuristics
- Skills filter with proficiency bar

## Deploy
Upload `index.html`, `styles.css`, `script.js`, and `README.md` to the root of your site repository or import the repository to Vercel. Add a custom domain through Vercel or GitHub Pages.

## Replace later
Update project card links to your actual GitHub repos after creating:

- `bace1-qsar-lab`

- `raredx-explainer`
